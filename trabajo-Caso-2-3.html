Especificar correctamente los modelos teóricos. 2. Ajustarlos en R respetando la estructura de cruce y anidamiento. 3. Verificar los Cuadrados Medios Esperados (CME). 4. Interpretar los resultados del ANOVA. --- ## Datos ```{r datos} campo <- data.frame( A = factor(rep(c("a1", "a2", "a3"), each = 24)), B = factor(rep(rep(c("b1", "b2", "b3", "b4"), each = 6), 3)), C = factor(rep(rep(c("c1", "c2"), each = 3), 12)), R = factor(rep(c("r1", "r2", "r3"), 24)), Y = c( 29.5,14.2,25.2,30.0,23.1,29.5,14.2,22.8,23.0,15.2,12.5,29.7, 24.3,28.2,23.9,29.7,28.1,29.6,13.5,25.5,16.0,29.7,21.9,13.8, 15.0,10.5,19.2,24.2,16.6,25.8,13.2,15.5,11.2,14.5,24.3,12.3, 24.9,10.2,23.2,28.3,11.6,25.5,12.4,18.9,28.3,25.1,11.9,23.2, 16.3,22.2,13.1,23.4,23.5,11.1,27.1,12.9,28.2,27.3,16.7,21.9, 28.1,29.6,16.6,25.5,22.9,19.1,18.7,22.4,26.7,25.4,16.4,22.0 ) ) ``` ```{r factores} A_fijo <- as.fixed(campo$A) # Caso 2 B_fijo <- as.fixed(campo$B) C_azar <- as.random(campo$C) # Caso 3 B_azar <- as.random(campo$B) C_fijo <- as.fixed(campo$C) Y <- campo$Y ``` --- ## Caso 2 – A cruzado, B y C anidados ### Modelo teórico $$ Y_{ijkl} = \mu + A_i + B_{j(i)} + C_{k(ij)} + \varepsilon_{l(ijk)} $$ No existen interacciones, porque los factores anidados no generan interacción con su anidador. ### Modelo en R ```{r caso2} modelo_caso2 <- lm( Y ~ A_fijo + B_fijo %in% A_fijo + C_azar %in% (A_fijo:B_fijo) ) anova(modelo_caso2) ``` ### Interpretación - **A (fijo):** evalúa diferencias entre los niveles principales del factor A. - **B(A) (fijo):** mide diferencias sistemáticas entre niveles de B dentro de cada nivel de A. - **C(AB):** cuantifica la variación aleatoria de C dentro de cada combinación A–B. **Conclusión Caso 2:** El análisis permite separar claramente la variabilidad sistemática (A y B(A)) de la variabilidad aleatoria (C(AB) y error), garantizando denominadores F correctos para cada prueba. --- ## Caso 3 – A y C cruzados, B anidado en A ### Modelo teórico $$ Y_{ijkl} = \mu + A_i + C_k + (AC)_{ik} + B_{j(i)} + (B(A)C)_{jk(i)} + \varepsilon_{l(ijk)} $$ Fuentes de variación: - A - C - A×C - B(A) - B(A)×C - Error ### Modelo en R ```{r caso3} m3 <- lm( Y ~ A_fijo * C_fijo + B_azar %in% A_fijo + C_fijo:(B_azar %in% A_fijo) ) gad(m3) estimates(m3) ``` ### Resultados del ANOVA A partir del ANOVA se observa que: - El efecto de A es (significativo / no significativo), indicando que existen (o no) diferencias entre niveles de A. - El efecto de C es (significativo / no significativo), indicando que existen (o no) diferencias sistemáticas entre niveles de C. - La interacción A×C evidencia que el efecto de C (cambia / no cambia) entre niveles de A. --- ## Interpretación del Caso 3 ### Factor A (fijo) - Evalúa diferencias sistemáticas entre los niveles principales de A. - Se prueba contra **B(A)**, porque B(A) representa la variabilidad interna de A. ### Factor C (fijo) - Evalúa diferencias sistemáticas entre los niveles del factor C. - Se prueba contra la interacción **B(A)×C**, ya que esta refleja la variabilidad de C dentro de cada nivel de A a través de B. ### Interacción A×C - Indica si el efecto de C cambia según el nivel de A. - Es clave para evaluar la estabilidad del efecto de C entre tratamientos principales. ### B anidado en A – B(A) - Mide diferencias entre niveles de B dentro de cada nivel de A. - Actúa como fuente de error para contrastar el efecto de A. ### Interacción B(A)×C - Evalúa si el efecto de C varía entre los niveles de B dentro de cada A. - Funciona como denominador natural para probar **A×C** y **C**. ### Cuadro teórico de estimates ```{r} ## Generación automática del cuadro teórico desde estimates() est <- estimates(m3) # ---- Tabla de estructura del diseño ---- tm <- as.data.frame(est$tm) tm$Fuente <- rownames(tm) tm <- tm[, c("Fuente", "A_fijo", "B_azar", "C_fijo", "n")] colnames(tm) <- c("Fuente", "A", "B", "C", "R") # ---- Tabla de Cuadrados Medios Esperados ---- mse_vec <- unlist(est$mse) mse <- data.frame( Fuente = rownames(tm), `E[CM]` = mse_vec, row.names = NULL ) # ---- Unión de ambas tablas ---- cuadro_teorico <- merge(tm, mse, by = "Fuente", all = TRUE) # ---- Orden correcto de las fuentes ---- orden <- c("A_fijo", "C_fijo", "A_fijo:C_fijo", "A_fijo:B_azar", "A_fijo:C_fijo:B_azar", "Res") cuadro_teorico <- cuadro_teorico[match(orden, cuadro_teorico$Fuente), ] # ---- Mostrar tabla final ---- knitr::kable(cuadro_teorico, caption = "Estructura teórica y Cuadrados Medios Esperados derivados automáticamente de estimates(m3)", align = "c") ``` ### Interpretación del cuadro teórico El cuadro obtenido a partir de `estimates(m3)` resume simultáneamente: - La estructura del diseño (columnas A, B, C y R). - Las fuentes de variación efectivas del modelo. - Las expresiones exactas de los Cuadrados Medios Esperados asociadas a cada fuente. Se observa que: - La fuente **A_fijo** incluye componentes asociados al efecto principal de A y a la interacción A×C, reflejando que el efecto de A puede verse modificado por el factor C y por la estructura anidada de B(A). - La fuente **C_fijo** recoge únicamente efectos sistemáticos debidos a C, confirmando su naturaleza de factor fijo en este diseño. - La interacción **A_fijo:C_fijo** refleja si el efecto de C cambia entre los niveles de A, constituyendo una fuente clave para evaluar la estabilidad del efecto de C. - La fuente **A_fijo:B_azar** recoge la variabilidad inducida por el anidamiento de B dentro de A, actuando como denominador para contrastar el efecto principal de A. - La interacción **A_fijo:C_fijo:B_azar** representa la variabilidad residual asociada a la interacción entre B(A) y C, funcionando como denominador natural para contrastar los efectos de C y A×C. - La fuente **Residuals** concentra exclusivamente la varianza experimental Ve, constituyendo el error puro del modelo. --- ## Coherencia con los Cuadrados Medios Esperados La función `estimates(m3)` reproduce exactamente los Cuadrados Medios Esperados del modelo mixto con A y C fijos y B aleatorio: - $$E[CM_A] = V_e + 6 V_{B(A)} + 24 \phi_A$$ - $$E[CM_C] = V_e + 3 V_{B(A)C} + 36 \phi_C$$ - $$E[CM_{AC}] = V_e + 3 V_{B(A)C} + 12 \phi_{AC}$$ - $$E[CM_{B(A)}] = V_e + 6 V_{B(A)}$$ - $$E[CM_{B(A)C}] = V_e + 3 V_{B(A)C}$$ donde: - $V_e$ es la varianza experimental, - $V_{B(A)}$ es la varianza del factor aleatorio B anidado en A, - $V_{B(A)C}$ es la varianza de la interacción B(A)×C, - $\phi_A$, $\phi_C$ y $\phi_{AC}$ representan los efectos fijos de A, C y A×C respectivamente. Esto confirma que: - C es tratado correctamente como **factor fijo**. - Los denominadores F están correctamente definidos. - El modelo en R coincide exactamente con el modelo teórico del **Caso 3** visto en clase. --- ## Conclusión general - La especificación correcta de **cruce** y **anidamiento** es fundamental para obtener pruebas F válidas. - En el Caso 3: - A se prueba con B(A) - C y A×C se prueban con B(A)×C El uso conjunto de: - `lm()` con operadores `*`, `%in%` y `:` - `gad()` para ANOVA mixto - `estimates()` para verificar CME permite validar completamente la coherencia estadística y teórica del análisis.
